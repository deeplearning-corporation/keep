# 贡献指南

感谢您对 KEEP 编译器的兴趣！虽然我们的编译器是闭源的，但我们欢迎社区反馈和问题报告。

## 报告 Bug

在报告 bug 之前，请：

1. 检查您使用的是最新版本的 KEEP 编译器
2. 搜索现有的 issue，确保该问题尚未被报告
3. 准备一个最小化的代码示例来重现问题

提交 bug 报告时，请包含：

- 您的操作系统和架构
- KEEP 编译器的版本
- 完整的命令行和输出
- 导致问题的源代码
- 预期行为和实际行为

## 功能请求

我们欢迎功能建议！请提供：

- 清晰的功能描述
- 使用场景和用例
- 为什么这对您有帮助
- 如果可能，提供代码示例

## 安全漏洞

如果您发现安全漏洞，请不要在 GitHub 上公开报告。请发送邮件至 security@keep-compiler.com。
